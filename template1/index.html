Horaires

Les entrainements pour les enfants ont lieu le lundi et le jeudi de 18h à 19h.

Pour les adultes, les entrainements ont lieu le lundi et le jeudi de 19h à 20h30. Pour les compétiteurs et les motivés un entrainement supplémentaire est prévu le mardi de 20h à 21h.

Tarifs

Une mensualité de 20 € pour les enfants.

Une mensualité de 25 € pour les adultes.
